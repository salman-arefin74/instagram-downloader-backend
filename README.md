# instagram-downloader-backend

## How to run:
* Run `npm install`
* Run `npm start`

It will run in the port 9001

## How to download:
* Go to an instagram post, the url will be like this: `https://www.instagram.com/p/C3X373nMFOB/`
* Copy the code of the post which in the sample URL is: `C3X373nMFOB`
* Go to `localhost:9001` and simply append `/download/C3X373nMFOB`
* So basically the URL: `http://localhost:9001/download/C3X373nMFOB`
* The image will be downloaded in your default download folder of the browser.
